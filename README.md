# 10 Print in Python 3 with Pygame
Re-creating 10 Print in Python 3 and Pygame

Inspired by the book 10 PRINT.

## Requirements

Pygame and Python 3

Install Pygame on any operating system with:

```bash
$ pip install pygame
```
